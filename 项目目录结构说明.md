# 乙羽工具箱 (Yiyu's Toolbox) 项目目录结构说明

为了保持项目根目录的整洁以及提高可维护性，本项目采用 **“核心与辅助分离”** 的目录设计方案。以下是各文件夹的分配及作用说明：

---

## 1. 📂 核心运行环境 (yiyu Toolbox core)
该文件夹是整个工具运行的心脏，包含了程序执行所需的所有实体文件。

*   **src/**: 核心 Python 源代码。
    *   `demo.py`: 程序主入口，管理界面加载。
    *   `*_processor.py`: 各功能的 AI 处理引擎逻辑。
    *   `*_ui.py`: 各功能的独立界面组件。
*   **model/**: AI 模型权重文件（`.pt`, `.pth`）。
*   **resources/**: 图片素材、图标文件、中间结果模板（`gray.png`, `mask.png`）。
*   **requirements.txt**: 运行环境所需的 Python 依赖列表。
*   **DLL 文件**: 运行所需的 `msvcp140.dll` 等动态链接库。

---

## 2. 📂 脚本库 (scripts)
该文件夹存放所有用于操作项目的工具脚本。

*   `run_source.bat`: **【常用】** 源码启动脚本。已完成路径重定向，双击即可启动核心目录中的程序。
*   `create_portable_version.bat`: 一键构建便携版程序的脚本。
*   `build_exe.bat`: 使用 PyInstaller 打包独立 EXE 的脚本。
*   `sync_github.bat` / `sync_mirror.bat`: 代码同步与备份工具。

---

## 3. 📂 技术文档 (docs)
存放项目的文字性资料和历史建模资产。

*   `CORE_ARCH_CN.md`: 核心架构开发指南。
*   `AI_READINESS.md`: AI 模型环境就绪手册。
*   `使用说明.txt`: 用户操作指南。
*   `icon图标建模模型/`: 存放图标设计的相关 3D 建模源文件。

---

## 4. 📂 发布中心 (releases)
存放所有对外的成品产物。

*   `yiyu_toolbox_portable/`: 当前版本构建出的绿色便携版文件夹。
*   `*.zip`: 各历史版本的压缩分发包。

---

## 5. 🏠 根目录文件
根目录仅保留最基础的项目标识文件。

*   `README_CN.md` / `README.md`: 项目主介绍及快速入门说明。
*   `.gitignore`: Git 忽略配置。
*   `项目目录结构说明.md`: (本文件) 记录当前的目录分配逻辑。
